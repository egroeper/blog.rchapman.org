ASA: DAP LUA function list

Sat, Nov 7, 2015
ASA 9.4.2 (which has Lua 5.0.2), AnyConnect 4.0 on iOS On the internets, I haven't been able to locate a list of Lua functions available for use with Dynamic Access Policies (DAP) on Cisco's ASA firewall. So I decided to just use Lua running on the ASA to tell me what functions and variables are in global scope. Not quite as good as documentation, but it works for now.
